# Windows 快速开始指南 ## 🚀 3 步快速安装 ### 第 1 步: 检查 Python 打开 PowerShell，检查 Python 是否安装： ```powershell python --version ``` ✅ 如果显示 `Python 3.8` 或更高版本，继续下一步。 ❌ 如果没有，从 [python.org](https://www.python.org/downloads/) 或 Microsoft Store 安装 Python。 ### 第 2 步: 运行安装脚本 ```powershell cd cli-agent python install.py ``` 等待安装完成（约 1-2 分钟）。 ### 第 3 步: 验证安装 **重新打开 PowerShell 或 CMD**，然后运行： ```powershell dnm --version ```
