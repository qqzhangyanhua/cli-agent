# _handle_file_reference

- 类型: function
- 来源: /Users/zhangyanhua/Desktop/AI/tushare/quantification/example/src/ui/input_handlers.py

## 摘要
处理文件引用输入（@ 开头）

职责：
1. 只输入 @ -> 交互式选择文件
2. @filename -> 快速选择文件
3. @filename operation -> 直接返回

Args:
    user_input: 原始输入（以 @ 开头）
    prompt: 提示符

Returns:
    完整的命令字符串，或 None（需要重新输入）
