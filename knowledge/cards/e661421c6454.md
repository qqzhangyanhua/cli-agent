# Code Review 功能完整实现指南 ## 📋 功能概述 基于 git diff 的智能代码审查功能，自动分析代码变更，发现潜在问题，并按严重性分级（🔴 严重、🟡 中级、🟢 普通）输出详细的审查报告。 ## 🎯 设计目标 1. **自动化审查**：无需手动指定文件，自动分析所有 git 变更 2. **智能分析**：使用 LLM 深度理解代码逻辑，发现潜在问题 3. **分级报告**：按严重性分级，帮助开发者优先处理关键问题 4. **可操作建议**：提供具体、可执行的改进建议 ## 🏗️ 架构设计 ### 核心模块 ``` code_review_tools.py # 核心实现模块 ├── analyze_code_changes() # 分析 git diff ├── perform_code_review_func() # 执行代码审查（LLM驱动） └── code_rev
